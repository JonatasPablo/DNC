                                        Responsabilidade com o CSS
Importância da Responsividade
Usuários cada vez mais exigentes
Uso de diversos dispositivos (notebooks, desktops, celulares, tablets)
Necessidade de criar sites prontos para diferentes resoluções
Soluções com CSS
Utilização de CSS para resolver problemas de responsividade
Uso de técnicas como Flexbox, Grid e Image Aquarius
Projeto Prático
Desenvolvimento de um site de uma construtora digital
Diferentes layouts para desktop, tablet e celular
Aprendizado de conceitos importantes de CSS, Flexbox, Grid e Image Aquarius
Ferramentas
Utilização do Figma para o projeto
Importância do Figma no desenvolvimento
Próximos Passos
Aprendizado sobre a interface do Figma
Exploração de funcionalidades como barras de ferramentas, camadas e protótipos
O objetivo deste módulo é capacitar os alunos a desenvolver sites responsivos e de alta qualidade, utilizando as principais técnicas e ferramentas do CSS. Através de um projeto prático, os alunos terão a oportunidade de aplicar os conceitos aprendidos e criar soluções adaptáveis a diferentes dispositivos.

Diferentes telas (desktop, tablet, celular)
Seções: apresentação, qualidades e processos, o que oferecemos, posts/blog, referências, contato, footer
Cada seção será uma <section> no HTML
Elementos de Texto
Títulos principais são <h1>, <h2>, <h3>
Tamanhos padrão, como lidar com eles no código
Posicionamento com Flexbox
Centralizar elementos na tela
Responsividade da imagem de fundo
Trabalhar com flex-direction para posicionamento horizontal e vertical
Grid Layout
Criar um grid responsivo
3 colunas no desktop, 2 colunas no tablet, 1 coluna no celular
Posicionar elementos dentro do grid
Flexbox Avançado
